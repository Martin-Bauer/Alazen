5: per "psql VGB_DB" steigt man in die VGB_DB ein
6: Hier kann man nun normale SQL Befehle eingeben und die Ergebnisse sich ausgeben lassen
7: KEINE ALTER TABLE BEFEHLE, die Verwaltung der DB übernehme ich
8: mit \q kommt man zurück in die normale Befehlszeile, per exit dann zurück zum luser
